안녕하세요 3조 중간발표를 맡게 된 배채은입니다
저희 조의 주제는 2D 데이터셋 압축에 따른 3D 비전 모델 성능 저하 분석 및 개선 방법입니다.
/
목차는 다음과 같습니다
/
먼저 수행배경 및 목표에 대해 간단하게 말씀드리겠습니다.
/
컴퓨터비전 분야의 다양한 작업들은 입력 소스의 품질에 민감하게 반응합니다.
그러나 대부분의 3D 연구에서는 압축되지 않은 고품질의 데이터셋을 사용해 성능을 보였습니다.
현실 세계에서 다뤄지는 이미지나 비디오는 모두 손실 압축이 진행된 상태이기 때문에, 입력 소스의 압축률과 3D 비전 모델의 성능 간의 상관관계를 분석할 필요가 있습니다.
/
따라서 저희는 총 3가지 목표를 잡았는데요,
첫번째는 압축이 영상 기반 3D 재구성 모델인 VGGT의 성능에 미치는 영향을 분석하는 논문을 작성하는 것,
두번째는 압축이 3D 렌더링 모델인 3D Gaussian Splatting의 성능에 미치는 영향을 분석하는 논문을 쓰는 것,
마지막으로 세번째는 압축에 따른 3D 모델의 성능 저하를 완화시킬 수 있는 방법을 찾아 소프트웨어 등록을 하는 것입니다.
현재는 두번째 목표까지 완료한 상태입니다.
/
다음으로 전체 시스템 구조도입니다.
/
세부적인 설명은 넘어가도록 하겠습니다.
/
다음은 진행상황입니다.
/
먼저 3DGS 파트의 진행상황인데요,
저희는 방송미디어공학회로 학회를 변경했으며, 추계학술대회 논문 제출을 완료하였습니다.
또한 이번주 금요일인 11월 14일에 열리는 학술대회 프로그램에서 포스터 세션에 배정받았고, 이를 위한 포스터 제작도 다 마친 상태입니다.
/
해당 페이지는 논문의 대표 그림입니다.
이미지 압축 코덱인 JPEG과 동영상 압축 코덱 HEVC를 사용해 압축을 진행하였고, 평가 지표로는 PSNR, SSIM, LPIPS를 사용하였습니다.
입력 데이터셋에 전처리 과정을 거친 후 압축하고, 3DGS 입력에 필요한 카메라 포즈 및 sparse point cloud를 추출하는 과정을 담았습니다.
/
이제 실험의 결론에 대해 설명드리도록 하겠습니다.
먼저 JPEG 압축 코덱을 적용하였을 때는 압축률 증가 시 정량적인 품질 저하가 비선형적으로 일어났습니다.
또한 일부 장면에서는 JPEG의 아티팩트가 3DGS 렌더링 과정을 거치면서 완화되어 SSIM 수치가 오히려 상승되는 경우도 있었습니다.
오른쪽 사진의 빨간색 박스를 보시면 압축 후 생긴 블록 아티팩트들이 렌더링 후에는 부드러워진 것을 확인할 수 있습니다.
/
다음으로 HEVC 코덱을 적용했을 때는 압축률이 증가할수록 비교적 점진적인 품질 저하를 보였으며, 일부 장면에서는 PSNR이 급격히 하락하는 이상치를 발견하기도 했습니다.
/
마지막으로 JPEG과 HEVC 두 코덱을 비교하였을 때, 비슷한 품질의 입력 이미지에서 JPEG 렌더링 결과가 HEVC보다 항상 더 높거나 같은 수준의 품질을 보였습니다.
이는 JPEG은 프레임 간 독립적인 압축을 시행하는 반면, HEVC는 인접 프레임을 참고한 예측 기반 압축으로 오차가 누적되었기 때문으로 해석됩니다.
/
논문을 기반으로 제작한 학회 포스터 최종본입니다. 넘어가도록 하겠습니다.
/
다음으로 VGGT 파트의 진행상황에 대해 설명하도록 하겠습니다.
VGGT도 마찬가지로 학회 논문 제출 및 포스터 제작까지 완료한 상태입니다.
/
논문의 대표 그림으로, VGGT에도 동일하게 JPEG 코덱을 사용하였으나 동영상 압축 코덱으로는 AVC 코덱을 사용하였습니다.
또한 평가 지표로는 Chamfer Distance와 AUC를 사용해 포인트 맵의 유사도와 카메라 포즈 추정 정확도를 측정하였습니다.
/
실험 결과입니다. 먼저 AVC와 JPEG 두 코덱 모두에서 압축률이 증가할수록 추정된 포인트 맵의 포인트 수가 줄어드는 경향을 확인할 수 있었습니다.
또한 PSNR 수치도 압축률이 증가할수록 하락하였습니다.
/
다음으로는 추정된 포인트 맵과 GT의 유사도를 Chamfer Distance를 써서 측정했는데요,
이 Chamfer Distance는 점수가 낮을수록 정답과 유사하다는 의미입니다. 
이때 평균적으로 압축률이 높을수록 유사도가 떨어지는 것을 확인했습니다.
그러나 laptop 카테고리에서는 예외적으로 강한 압축에서 유사도가 오히려 증가하는 경우가 있었는데요, 이는 3D 장면 추론 모델이 빛 반사가 있는 입력을 받았을 때 깊이를 정확하게 예측하지 못하는 현상 때문인 것으로 해석됩니다.
/
마지막으로 AUC를 사용해 VGGT의 카메라 포즈 예측 성능을 측정해봤을 때, AVC 코덱의 경우 압축률이 높을수록 예측 성능이 떨어졌으나
JPEG 코덱에서는 유의미한 영향을 주지 않았습니다.
이는 JPEG의 세부적인 디테일을 저하시키는 압축 방식이 이미지를 전역적으로 해석하는 VGGT에는 큰 영향을 주지 않은 것으로 추정되었습니다.
/
마찬가지로 제작 완료된 포스터 사진입니다.
/
이제 성능 개선 연구에 대한 진행상황을 설명드리겠습니다.

mipnerf360의 4가지 씬에 대해 JPEG 90, 50, 10 세 가지 압축 강도를 적용해서 실험을 진행할 예정입니다.
