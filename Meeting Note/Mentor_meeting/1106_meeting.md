# 11.06 멘토 미팅 회의

## 전처리 모델 관련
- 찾아온 모델들 그대로 진행해도 좋을 것 같음
- 선택한 세 개(EARN, DriftRec, LIPT) 뿐 아니라 이중 JPEG 압축 중점인 OAPT도 같이 진행해도 좋을 것 같음

## 실험 관련
- 전처리 모델들 간 속도나 성능 차이를 유심히 고려하기 보다는 체크하고 결과 리포트만 잘 하면 됨
- 멘토님 의견으로는 생각만큼 성능이 잘 안 나올 것 같다고 하심

## 학술대회 포스터 관련
- 3DGS 포스터에서 전체적인 틀이나 구성은 괜찮음
- 텍스트를 좀 줄이고 그림이나 결과 사진을 잘 보이게 확대 배치하는게 좋음
- VGGT 포스터는 전반적으로 괜찮음
