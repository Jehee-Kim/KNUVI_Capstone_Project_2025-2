# 09.24 멘토 미팅 회의록

## 1. VGGT 실험 관련 피드백
- 압축이 성능에 미치는 영향을 확인하려면, 코덱과 QP를 프레임별로 랜덤하게 적용하는 기존의 방식으로는 실험의 정당성을 확보하기 어렵다. 기준이 다른 것끼리 비교하는 셈.
- 따라서 같은 프레임에 여러 종류의 코덱과 파라미터를 적용해야 한다.

## 2. 3DGS 실험 관련 피드백
- 결과 중간중간에 재검증이 필요한 부분이 한 두 곳 있는데, 다시한번 확인해볼 것.

## 3. 논문 작성 관련 피드백
- 초록은 5문장, 길어도 6문장으로 마무리 짓는 것이 좋다.
- 추후에 어느 정도 완성이 되고, 대학원생이나 박상효 교수님을 거친 후 한번 피드백을 줄 수 있을 것 같다.
- 3DGS/VGGT 파트 모두 기존에 이러한 부분을 체계적으로 연구한 논문이 거의 없었다는 점을 어필하면 좋을 것 같다.

## 4. 이후 할 일 정리
- VGGT 실험 재설계: 3DGS 실험 설계를 참고. 멘토님께 피드백을 받아도 좋음
    - 실험에 사용한 코드 대학원생분들께 크로스체크 받아야 함
- 3DGS 실험 계속 진행: HEVC 적용한 결과 도출하기
- 논문 작성: 대학원생분들 피드백 적용해서 수정하기
