**09.17 멘토 미팅 피드백 정리**

## 1) 2D 파트 관련

- Co3D에 포인트 클라우드 GT가 있으니 이를 활용해 VGGT로 생성된 포인트 클라우드와의 비교를 중점으로 가져가는 게 좋을 것 같다.
    - 이후에 Downstream Tasks는 부가적인 평가지표로 사용하는 것이 좋아보임.
- 압축률, 코덱을 랜덤하게 설정하기보다는 균등하게 적용해서 경향성을 파악하는 게 더 좋을 것 같다.

## 2) 3D 파트 관련

- PSNR 스코어는 원본 GT와 압축 렌더링 결과를 비교해야 한다.
- 입력 데이터 간, 원본GT와 압축 렌더링 결과 간 PSNR도 추가해서 비교할 수 있도록 할 것.
- HEVC 뿐 아니라 JPEG 코덱도 사용하고 QP도 4개 정도로 늘리면 좋을 것 같다.
    - 리뷰어 입장에서는 HEVC만 사용했으니 JPEG를 적용한 결과도 궁금할 것이기 때문

## 3) 기타 피드백

- 2D/3D파트 명칭이 조금 안 맞는 것 같다.
    - 이미지/비디오 코덱 차이, 모델 차이기 때문에 명칭을 고민해보는 것이 좋을 것 같다.
- 멘토님께서 보내주신 논문에서 실험 조건이나, 결과를 어떤 식으로 도출하는지를 볼 것.
    1. 압축된 걸 렌더링했을 때 원본 GT 대비 얼마나 떨어지는지 비교하는 방법
    2. 원본 GT가 없는 경우 압축한 영상을 가GT로 만들어서 비교하는 방법
- 대략적으로 결과를 어느정도 상정해두고 이를 기대하면서 실험을 진행하면 좋다.
    - 이런 논문은 정보를 제공하는 것이다보니 실험이 많아야 한다.
- 논문 작성 관련해서, 초록 작성은 다른 논문들을 참고해보면 좋을 것 같다.
