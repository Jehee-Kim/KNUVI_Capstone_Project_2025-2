# 10.15 멘토회의

### I. 3DGS 파트
* 표 관련 피드백
  * 동일하거나 비슷한 내용의 표는 반복해서 사용하지 않음
  * 코덱 간의 비교 결과를 제시할 때는 표 대신 그래프를 활용할 것
  * 표를 사용할 경우, 세부 데이터 대신 평균값 등 핵심적인 결과 위주로 정리

### II. VGGT 파트
* Chamfer Distance
    * CD 결과를 봤을때 변화가 미미한 부분이 있지만 전체적으로 성능 변화가 있었다고 서술해도 괜찮음
    * Origin CD 해석을 바탕으로 장면의 특성과 압축 정도에 따른 CD 변화 경향성에 대한 해석을 추가해도 좋을 것 같음
    * 데이터셋별 원인 분석을 GPT를 활용해 하는 것도 좋음, 시간상 어렵다면 후속 연구로 고려
* AUC 해석 보강
    * AVC는 성능이 저하되는데 JPEG은 성능이 저하되지 않는 이유에 대해 분석해보면 좋음
* 그림은 출력되지 않는데 테이블에는 값이 있는 부분 확인할 것

### III. 전체 코멘트
* 전반적으로 비디오 코덱을 적용한 경우의 점수가 더 안 좋은 경향이 나타나는데, 이에 대한 근본적인 원인 분석이 들어가면 좋을 것 같음 (시간이 부족하면 후속 연구로 고려)
