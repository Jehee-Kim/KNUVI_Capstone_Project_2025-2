Core Technologies

3D Gaussian Splatting

An explicit radiance field representation using anisotropic 3D Gaussians. We analyze how compression affects the initialized point cloud quality and final rendering.

VGGT (Visual Geometry Grounded Transformer)

A transformer-based approach that leverages deep learning to understand 3D geometry from 2D inputs. We test its robustness compared to traditional optimization methods.

COLMAP Automation

Automated Structure-from-Motion pipeline handling feature extraction, matching, and sparse reconstruction.

Evaluation Metrics

  • PSNR (Peak Signal-to-Noise Ratio)

    Image quality metric measure pixel-level fidelity.

  • SSIM (Structural Similarity Index)

    Measures perceived quality degradation.

  • LPIPS (Learned Perceptual Image Patch Similarity)

    Deep learning based metric matching human perception.

  • Chamfer Distance

    Geometric accuracy of point clouds.

  • AUC (Area Under Curve)

    Aggregate score for point cloud accuracy thresholds.

  • Point Count

    Number of valid 3D points in the final reconstruction.
